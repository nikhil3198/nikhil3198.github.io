Projects

"Dream it. Believe it. Achieve it."

Stock Market Analysis

The project aimed to analyze the inter-dependencies between stock market of US companies and the trends in popularity of the Twitter hashtags. It includes interesting visualizations and various models are applied in order to forecast the stock market prices.



Word - Phonetic Transcription

Translation of words into corresponding phonetics using Seq-Seq LSTM RNN model, Embedding model and Attention model and comparison of the generation of phonetic sequence by the appropriate models was achieved.
Attention model + Beamsearch turned out to be better model for word - phonetic transcription.

Fashion Apparel Classification

A combined model called MaskRCNN was applied using Transfer Learning approach on iMaterialist Fashion and Paperdoll Dataset. The model generates bounding boxes and segmentation masks for each instance of an object in the image and also classifies various classes associated with the object.

Image - Image Translation

Comparative study towards general approach of image to image translation using Vanilla GAN and Conditional GAN is implemented. CGAN comes out to be the promising approach for various image to image translation tasks which includes sketch-image translation, image colorization, etc..

SelfLess Acts

Micro-service model of photo sharing Web App hosted on AWS cloud is developed using Django framework, SQLite3 Database system and Linux Docker-containers. REST API is implemented on Back-End along with Load Balancing, Auto Scaling and Fault Tolerance features to make the application more flexible.

KonfBot

An interactive Chatbot called KonfBot was built to show various tech conferences happening across the world. It was developed using Azure Bot Service with NodeJS and LUIS.ai framework to understand the intents and entities. 




ID Management System

The project gives information about person identification details. The key features includes various ID's of a person such as Aadhaar, DL, Voter ID, Passport, Ration card and Insurance details. It's developed using php-MySql Database, and frontend includes HTML, CSS3.




IOT Weather Station

The IOT based project aims to show the current Humidity, Temperature and Pressure parameters on the LCD as well as on the internet server using Rasberry Pi. All this data is sent to ThingSpeak server for live monitoring from anywhere in the world over internet and finally ThingSpeak analyses the data and shows it in a Graphical form.



Saastha Educare

Developed an educational website for an institute. The site give necessary details of institute, facilities provided, course info, different branches located, admission process, etc. The technologies we used for achieving this project are HTML5, CSS3, Javascript, PHP, and Bootstrap framework. The site is highly responsive which can be viewed in any screen width.
